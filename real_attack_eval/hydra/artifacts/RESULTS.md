## Key Findings

- AUC drop on real traces is explained by **software/kernel version shift** (10-year gap: ADFA-LD 2011 vs Debian Bullseye 2021)
- `clock_gettime` (24.5% → 0%) and `poll` (17.9% → 1.8%) disappear; `openat` and `mmap2` dominate modern traces
- SSH Win-DR lower (58.6%) due to DH key exchange creating many crypto I/O windows that appear benign
- **Trace-level detection 100%** on real attacks despite domain shift — model still reliably identifies attack sessions
